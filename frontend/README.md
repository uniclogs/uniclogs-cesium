### Cesium Front-End Application

```bash
cd frontend
```

Install `nodejs` and `yarn` for your system

```bash
sudo apt install nodejs yarnpkg
```

Install JavaScript dependencies

```bash
yarn install
```

Start the app

```bash
yarn dev
```

Goto `localhost3000` in a web browser
